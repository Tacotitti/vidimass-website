## 🎨 Quick Customization

### Change Colors (Easy!):
Open `index.html` and find/replace:

```html

from-violet-600 to-pink-600

<!-- Change to: Blue-Teal -->
from-blue-600 to-teal-600

<!-- Or: Orange-Red -->
from-orange-600 to-red-600
```

## 🐛 Troubleshooting

### Page looks broken?
**Solution:** Make sure you have internet connection!  
(Tailwind CSS and Google Fonts load from CDN)

### Animations not working?
**Solution:** Try a modern browser (Chrome, Firefox, Safari, Edge)  
(IE11 not supported - uses modern CSS)

### Mobile menu not appearing?
**Solution:** Check if JavaScript is enabled in your browser

### Orbs causing lag?
**Solution:** On older devices, this is normal  
(Blur effects are GPU-intensive)
